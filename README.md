# OpenGL_Win32_3D_Cube_Renderer

This project demonstrates rendering of a 3D cube using the OpenGL programmable pipeline in C.  
The application creates a Win32 window and manually initializes the OpenGL rendering context without using external libraries such as GLFW or GLUT.

The goal of this project is to understand the fundamentals of the modern graphics pipeline and low-level graphics programming.

## Features

- 3D Cube Rendering
- OpenGL Programmable Pipeline
- Vertex and Fragment Shader implementation using GLSL
- Matrix transformations (Model, View, Projection)
- Win32 API window creation and message handling
- GPU buffer management using VAO and VBO

## Technologies Used

Language  
C,C++

Graphics API  
OpenGL (Core Profile)

Platform  
Windows (Win32 API)

Shader Language  
GLSL

## Graphics Concepts Implemented

- Vertex Buffer Objects (VBO)
- Vertex Array Objects (VAO)
- Shader compilation and linking
- Model, View and Projection matrix transformations
- Real-time rendering pipeline

## Learning Objective

This project focuses on understanding low-level graphics programming including:

- OpenGL rendering pipeline
- GPU data management
- Shader programming
- Matrix mathematics for 3D graphics
- Platform specific window and context creation
